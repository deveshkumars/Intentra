## Decisions

- **All Markdown, no JSON:** Originally had an IntentSchema JSON file. Removed it. If it's not readable prose, it doesn't belong in `.intentra/`.
- **Exact prompts, not summaries:** The Prompt section must be the literal words the human typed. This is the raw intent, preserved verbatim.
- **One file per session:** Each handoff contains prompt + plan + state in one doc.
- **Immutable snapshots:** Never edit an existing handoff. Write a new one.
