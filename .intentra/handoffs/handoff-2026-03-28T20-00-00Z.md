Implement the "Stateful Markdown handoffs" element from the Intentra masterdoc:
a standardized resume file that makes handoffs between humans and agents lossless.

plans + handoff docs. Everything in Markdown. No JSON. No schemas. Just readable
documents that any agent or human can pick up and run with.

## Decisions

- **All Markdown, no JSON:** The intent schema was originally JSON. Removed it. If it's not readable prose, it doesn't belong in `.intentra/`. English is the format.
- **One file per session:** Each handoff is a single Markdown doc containing prompt + plan + state. No splitting across multiple files.
- **Immutable snapshots:** Never edit an existing handoff. Write a new one. The file documents what was true at a point in time.
- **Four skill modes:** Write, resume, list, check. Covers the full lifecycle from creating a handoff to verifying it's still valid.
