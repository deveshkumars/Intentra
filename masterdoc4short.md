Intentra is an infrastructure layer designed for engineering teams running multiple AI agents simultaneously. It addresses three core gaps in current agentic development workflows: the loss of 'why' (prompts and reasoning disappear after a chat session closes), the inability to supervise agents without being desk-bound, and the absence of cultural guardrails that prevent agents from producing technically correct but organizationally wrong outputs. The system captures intent as append-only Markdown artifacts in a `.intentra/` directory (PROMPTS.md, PLANS.md, HANDOFFS.md), enforces team culture via a machine-readable `culture.json` and runtime safety hooks (`/careful`, `/freeze`), and streams real-time agent activity to a mobile app via SSE. The core technical architecture consists of a Bun HTTP server with a 200-event ring buffer, three ingestion paths (JSONL file watcher, POST /progress, and a PostToolUse hook), and a React Native/Expo mobile client with exponential backoff reconnection, event deduplication, and parallel backfill on reconnect. The team claims significant portions of this are already shipped: the SSE pipeline, JSONL watcher with a 3-tier fallback, mobile dashboard, resilient client hook, tracked agent CRUD with SSE broadcast, and safety hook telemetry (with 20+ real `hook_fire` activations logged in a single development session). The `.intentra/` artifact model and `/handoff` skill are also listed as complete. Intentra positions itself not as an agent or IDE tool, but as the coordination layer between intent and execution — analogous to how Git unified content-addressable storage into a protocol. The team is building on top of their existing gStack skill system (Claude Code-based), which already emits structured telemetry. The differentiation thesis is that culture enforcement as a runtime primitive, in-repo durable intent artifacts, and cross-tool mobile observability are structurally difficult for incumbents (GitHub, Devin, Cursor) to replicate because it contradicts their product architectures.
